Tentang Saya

Halo! Saya adalah mahasiswa semester 3 Jurusan Ilmu Komputer di Universitas Yatsi Madani. Meskipun saya belum memiliki pengalaman kerja dalam bidang komputer, saya memiliki keinginan untuk mendalami lebih jauh bidang ini dan terus mengembangkan keterampilan yang saya pelajari di bangku kuliah. Saat ini, saya fokus untuk memahami konsep-konsep dasar dalam ilmu komputer, seperti pemrograman, algoritma, dan struktur data. Saya berusaha untuk memperdalam pengetahuan saya melalui materi kuliah. Meskipun saya masih dalam proses belajar dan belum merasa ahli, saya berkomitmen untuk terus berkembang dan saya percaya bahwa setiap pengalaman baru adalah kesempatan untuk meningkatkan keterampilan diri dan berharap dapat memperoleh kesempatan yang dapat membantu saya tumbuh lebih lanjut dalam bidang ini.
